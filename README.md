# Bulls cows vue parcel

## Environment

- Node v16.20.2
- npm 8.9.4
- yarn 1.22.22

## Application

### Prepare project

* clone this repo

### Install application

Run `npm i` or `yarn`

### Run application

Run `npm run dev` or `yarn dev`

## Frontend

Frontend project is placed in `src/web` directory

### Dev

Run `npm run dev` `yarn dev`
